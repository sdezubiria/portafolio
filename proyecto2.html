Portfolio Path

Diseño web y branding para una marca de asesoría de arte

Portfolio Path es un proyecto de branding y diseño web desarrollado para un servicio de asesoría en arte. El enfoque principal fue construir una identidad visual cercana y cálida, que transmitiera confianza y acompañamiento en el proceso creativo. Para lograrlo, se incorporaron elementos del arte tradicional —texturas, trazos manuales y composiciones inspiradas en materiales analógicos— tanto en la marca como en la experiencia digital. El resultado fue una plataforma visualmente rica, accesible y coherente con los valores del proyecto: sensibilidad artística, guía personalizada y conexión humana.
